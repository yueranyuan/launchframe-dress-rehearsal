# Launchframe Practice — Launch-Day Runbook

One operator. Keep this executable.

Hard rule: never solicit upvotes.

Status: dress rehearsal. Execute as a tabletop run; do not perform external publish/post actions.

## Go / No-Go Preflight

- [x] Public rehearsal repo created.
- [ ] License selected and exact text committed.
- [ ] SECURITY, CONTRIBUTING, CODE_OF_CONDUCT, issue templates, privacy/terms/telemetry docs are filled or marked not applicable.
- [x] Rehearsal install command published and verified.
- [x] Demo works from local rehearsal artifacts.
- [x] Repo collateral is rehearsal-ready.
- [x] Package path is rehearsal-ready.
- [ ] README and social image are final.
- [ ] Website URL, canonical URL, OG image, and analytics are verified. Current rehearsal URL is HTTP until GitHub Pages certificate issuance completes.
- [ ] Founder/operator can monitor for 4-6 hours.

## T-3 Days

- [ ] Newsletter heads-ups.
- [ ] Builder DMs.
- [ ] Awesome-list PRs, if public repo exists.
- [ ] Final demo asset.
- [ ] Final data scrub.

## T-1 Day

- [ ] Cold-machine dress rehearsal.
- [ ] Finalize HN title and first comment.
- [ ] Finalize social and Reddit drafts.
- [ ] Confirm launch window.
- [ ] Confirm no vote-solicitation language exists anywhere.

## T-0

- [ ] Final smoke test.
- [ ] Publish package, if applicable.
- [ ] Flip/publish repo.
- [ ] Verify website/metadata/analytics.
- [ ] Post Show HN.
- [ ] Post founder first comment.
- [ ] Publish social thread.
- [ ] Monitor HN and issues.
- [ ] Post Reddit natively after a delay.

## T+1 To T+7

- [ ] Publish technical article.
- [ ] Keep responding to issues/PRs.
- [ ] Track metrics.
- [ ] Write retro.
- [ ] Feed lessons back into `../../playbook/`.

## Rollback

- Broken install: unpublish/deprecate bad package version if registry rules allow; pin README to known-good path; update HN comment with honest status.
- Bad bug: open public issue, document workaround, patch quickly, do not hide the failure.
- Accidental leak: take repo/package/site down if needed, rotate secrets, publish a clear postmortem if public users were affected.

## Current Blockers

- No public repo URL.
- No package/install path.
- No domain/hosted site.
- No selected license.
- No security/support contacts.
- No public policy docs.
- No production social/OG assets.
