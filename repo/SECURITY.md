# Security Policy

Status: dress rehearsal.

## Supported Versions

| Version | Supported |
|---|---|
| `practice-2026-06-15` | Yes, for rehearsal review only |

## Reporting A Vulnerability

Please report security vulnerabilities to:

- Email: `security@example.com`
- Advisory URL: `https://github.com/yueranyuan/launchframe-dress-rehearsal/security/advisories`

Do not open a public GitHub issue for a suspected vulnerability.

## Response Expectations

- Initial acknowledgement target: 2 business days.
- Triage/update target: 7 business days.
- Disclosure policy: coordinated disclosure after fix.

## Scope

In scope:

- Launchframe repo templates.
- Launchframe static site.
- Launchframe generated launch artifacts.

Out of scope:

- Third-party services not controlled by the project.
- Social engineering.
- Denial-of-service testing.
- Rehearsal placeholder domains, packages, and community URLs.
